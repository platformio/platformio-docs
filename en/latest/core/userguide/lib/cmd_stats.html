pio lib stats

Deprecated since version 6.0:

Warning

This command is deprecated and will be removed in the next releases.

Please use the unified Package Management CLI instead.

Usage

pio lib stats

Description

Show PlatformIO Library Registry statistics:

  • Recently updated

  • Recently added

  • Recent keywords

  • Popular keywords

  • Featured: Today

  • Featured: Week

  • Featured: Month

This information is the same that is shown on this page:

  • https://platformio.org/lib

Options

--json-output

Return the output in JSON format
