Usage

pio [OPTIONS] COMMAND

Options

--no-ansi

Do not print ANSI control characters.

See also PLATFORMIO_NO_ANSI and PLATFORMIO_FORCE_ANSI environment variables.

--version

Show the version of PlatformIO

--help, -h

Show help for the available options and commands

$ pio --help
$ pio COMMAND --help
