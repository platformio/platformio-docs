.. _cmd_org_destroy:

pio org destroy
===============

.. versionadded:: 5.0

.. contents::

Usage
-----

.. code-block:: bash

    pio org destroy ORGNAME

Description
-----------

Destroy an organization and its teams.

If you need to remove a user from an organization, please use :ref:`cmd_org_remove` command.
